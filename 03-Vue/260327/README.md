## 1. 📖 배운 내용
- **주제**: Composition API
- **단어**:
- <Kbd>setup()</kbd>: data, method, domputed 옵션이 사라자고 **setup()** 에 모든 데이터를 입력
- <Kbd>ref()</kbd>:변수 하나에 값이 1개 들어갈 때 사용(원시데이터) ex)숫자형, 문자형, boolrap(논리형)
- <Kbd>reactive()</kbd>:파생데이터 ex)배열(인덱스 접근), 객체리터럴(키 접근)
- <Kbd>computed()</kbd>: 값을 자동으로 계산하고 싶을 때 사용
- <Kbd>watch()</kbd>: 변경된 값, 변경되기 전 값을 처리하고 싶을 때/ 감시대상이 변경되면 비동기 통신 같은 복잡한 처리를 하고 싶을 때 사용
- <Kbd>watchEffect()</kbd>: watch를 사용하면 화살표 함수로 지정해줘야하는 번거로운을 없애기 위해 watchEffect 안에 있는 건 다 감시대상으로 자동인식
- <Kbd>화살표 함수</kbd>: function 생략, () => {} / (재료) => {실행 동작}
- <Kbd><script setup></kbd>: 
- **중요**: 오늘 특히 중요하다고 강조된 부분
