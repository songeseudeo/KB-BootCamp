## 1. 📖 배운 내용
- **주제**: 컴포넌트 심화
- **중요**: 슬롯(구멍)
  **추가 설명**
  - 붕어빵 틀을 부모에게 준다. 부모는 그 안에 팥을 넣을지 슈크림을 넣을지 내용물만 결정해서 다시 자식에게 줍니다. (디자인은 이미 붕어빵 모양으로 정해져 있음)
  - 자식이 부모에게 지금 반죽이 100g 남았다 라는 데이터를 알려준다. 그럼 부모는 그 데이터를 보고 그럼 그 반죽으로 붕어빵을 만들지 말고, 하트 모양 와플(템플릿)로 구워 라고 데이터에 맞춰 디자인까지 바꿔버리는 것
- **단어**
- <Kbd>슬롯</kbd> : 자식의 데이터(내용)와 부모의 템플릿(디자인)을 결합하여, 상황에 맞는 화면을 동적으로 구성하고 싶을 때 사용
- <Kbd>텔레포트</kbd> : 컴포넌트의 논리적 구조는 유지하면서, HTML 요소를 DOM의 다른 위치(예: body)로 렌더링하는 기능

- **v-model**: 서로 다른 컴포넌트가 데이터를 공유하는 법
