Vue 실무 구분 예제

computed는 계산된 값을 보여줄 때, method는 버튼/이벤트로 동작을 실행할 때, watch는 값 변화에 반응해 API 같은 후속 처리를 할 때 사용합니다.

아래 화면은 세 가지를 한 페이지에서 비교한 구현 예제입니다.
왼쪽은 합계 자동 계산, 가운데는 버튼 클릭 시 증가, 오른쪽은 검색어 변경 감지 + debounce 로그를 보여줍니다.
computed

계산된 값

화면에 보여줄 파생 데이터를 만들 때 사용합니다. 아래 예제는 x와 y가 바뀌면 합계가 자동으로 다시 계산됩니다.
자동 계산 결과
{{sum}}
핵심: 보여줄 값을 계산할 때 적합합니다.
method

동작 실행

버튼 클릭, 저장, 삭제처럼 사용자가 어떤 행동을 했을 때 실행하는 일반 함수입니다.
현재 카운트:
{{count}}
핵심: 버튼 누르면 실행되는 행동에 적합합니다.
watch

변화 감지

값이 바뀌는 순간 후속 작업을 할 때 사용합니다. 아래는 검색어 변화를 감지하고 debounce 후 로그를 남기는 예제입니다.
현재 검색어: {{keyword}}
watch 로그
{{logsText}}
핵심: 값이 바뀌었을 때 뒤에서 할 일에 적합합니다.
