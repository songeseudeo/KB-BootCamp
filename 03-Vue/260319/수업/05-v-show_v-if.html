Vue3 조건 렌더링 실습

1. v-show

입력값 : {{ amountShow }}

-100 입력 시 아이콘이 보이고, 100 입력 시 아이콘이 숨겨지는지 확인한다.

2. v-if

입력값 : {{ amountIf }}

-100 입력 시 아이콘이 생성되고, 100 입력 시 DOM에서 제거되는지 확인한다.

3. v-if / v-else-if / v-else

회원님의 등급 : Gold Silver Bronze Basic

100000, 300000, 700000, 1500000을 각각 입력해서 Basic → Bronze → Silver → Gold 순서로 바뀌는지 확인한다.

4. v-show 와 v-if 비교

공통 조건: 값이 0보다 작으면 경고 표시

v-show 방식: 경고 표시
v-if 방식: 경고 표시

개발자도구로 확인하면 v-show는 요소가 DOM에 남아 있고, v-if는 조건이 거짓일 때 요소가 제거되는 것을 볼 수 있다.

5. template + v-if

체크 시 여러 요소가 함께 나타나고, 체크 해제 시 함께 사라지는지 확인한다.
