Vue3 v-for 실습

1. 배열 반복

번호 이름 전화번호
{{ contact.no }} {{ contact.name }} {{ contact.tel }}

contacts 배열의 길이만큼 행이 생성되는지 확인한다.

2. 객체 반복

선택된 값 (코드값) : {{ selectedRegion }}
선택된 값 (텍스트) : {{ selectedRegion2 }}
regions 객체 : {{ regions }}

객체의 key는 option의 value로, value는 화면 표시 텍스트로 사용되는지 확인한다.

3. index 사용

순번 no 이름 전화번호
{{ index + 1 }} {{ contact.no }} {{ contact.name }} {{ contact.tel }}

실제 데이터 번호(contact.no)와 화면용 순번(index + 1)의 차이를 확인한다.

4. template v-for

번호 이름 전화번호

4개마다 구분선이 들어가고, 여러 요소를 한 번에 반복할 때 template을 사용할 수 있음을 확인한다.
