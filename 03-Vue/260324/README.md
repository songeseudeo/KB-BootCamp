# 📅 2026-03-24 학습 기록

## 1. 📖 배운 내용
- **주제**: 단일 파일 컴포넌트를 이용한 Vue 애플리케이션 개발
- **중요**: 1. 컴포넌트 나누기, 2. props, 3. emit

### 🛠️ 프로젝트 만들기
1. `npm init vue` : vite 기반으로 프로젝트 구성
2. `cd test-vue-add`
3. `npm install`
4. `npm run dev`

### 📦 컴포넌트 등록 및 지정
1. **전역 방법**: 한 번만 등록하면 바로 사용 가능 (사용하지 않는 컴포넌트도 포함) [`전역 방법`](./전역%20방법)
2. **지역 방법**: 사용하는 컴포넌트 내에서 매번 등록 (권장) [`지역 방법`](./지역%20방법)
3. **컴포넌트 지정**: `<script/>`, `<template/>`, `<style/>` 구조로 작성
4. **옵션 설정**: `components` 옵션에 사용할 컴포넌트 지정 [`컴포넌트 지정`](./컴포넌트%20지정)

### 📤 컴포넌트 정보 전달
* **props**: 부모 -> 자식 방향으로 데이터 전달
* **emit**: 자식 -> 부모 방향으로 이벤트 전달
* **중요**: 데이터 흐름은 단방향으로 유지됨

## 3. 🚨 오류 및 해결 (Troubleshooting)
- **문제**: 지역 등록 시 `defined but never used` 에러 발생
- **원인**: 컴포넌트를 import하고 등록했지만 `<template>`에서 태그로 사용하지 않음
- **해결**: `<template>` 내에 `<컴포넌트명 />` 태그를 추가하여 해결
