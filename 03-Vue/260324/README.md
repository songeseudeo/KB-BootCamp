## 1. 📖 배운 내용
- **주제**: 단일 파일 컴포넌트를 이용한 Vue 애플리케이션 개발
- **중요** 1. 컴포넌트를 나누는 방법/규칙 2. props 3. emit
  
- **프로젝트 만들기:**
1. npm init vue : vite 기반으로 프로젝트 구성
2. cd test-vue-add
3. npm install
4. npm run dev
   
- **컴포넌트 등록:** 
- <Kbd>전역 방법</kbd> : 한 번만 등록하면 바로 사용 가능 (사용하지 않는 컴포넌트에서도 포함) [`전역 방법`](./전역%20방법)
- <Kbd>지역 방법</kbd> : 사용하는 컴포넌트내에서 매번 등록 (권장) [`지역 방법`](./지역%20방법)
- **컴포넌트 지정:** <script/> <template/> <style/>
- [`컴포넌트 지정`](./컴포넌트%20지정) components옵션에 컴포넌트 지정
- **컴포넌트 정보 전달:**
- <Kbd>props</kbd> : 부모 -> 자식 
- <Kbd>emit</kbd> : 자식 -> 부모
-  **중요** 하나의 컴포넌트에서 하나의 컴포넌트만 전달 가능
