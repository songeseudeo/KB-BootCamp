## 3. 🚨 오류 및 해결 (Troubleshooting)
- **문제**: 데이터는 분명히 존재하는데 화면에 리스트가 전혀 나타나지 않음.
- **원인**: App.vue에서 정의한 데이터 이름(todoList)과 템플릿에서 사용한 이름(todolist)이 대소문자 하나 차이로 달랐음.
- **해결**: 모든 파일에서 변수명을 **todoList**로 통일하여 해결.
