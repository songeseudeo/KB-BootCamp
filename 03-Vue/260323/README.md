# 📅 2026-03-23 학습 기록

## 1. 📖 배운 내용
- **주제**: Vue 생명주기, 이벤트 처리, 스타일 적용, TodoList 예제
- **단어**
- <kbd>Vue 생명주기</kbd> : Vue 컨포넌트가 생성되고, 보여지고, 업데이트 되고, 사라지는 모든 과정
- <kbd>@이벤트</kbd> : 기본 형식 v-on:[이벤트 이름]
- 
- **함수**: 개념 요약
- <kbd>created()</kbd> : Vue 인스턴스를 생성 후 동기적으로 호출 (API 호출, 초기값 설정)
- <kbd>mounted()</kbd> : Dom Tree 생성 후 연결
- <kbd>updated()</kbd> : 반응하고 처리하는 모든 과정 자동으로 호출(수정) 
- <kbd>unmounted()</kbd> : 연결 해제, 모든 이벤트 제거
- <kbd>e.target.value</kbd> : input에 한글이 있을 경우 모든 글자를 다 가져옴
- <kbd>cancelable</kbd> : 기본 이벤트의 실행 취소를 할 수 있는지 여부
- <kbd>@input</kbd> : 안에는 무조건 화살표 함수
- <kbd>key</kbd> : 이벤트를 발생시킨 키코드 이벤트, 내가 어떤 걸 눌렀는지 알 수 있음
- <kbd>button</kbd> : 이벤트를 발생시킨 마우스 이벤트, 마우스 오/왼 버튼, 마우스 휠
- <kbd>preventDefault()</kbd> : 기본 이벤트의 자동 실행 막기
- <kbd>stopPropagation</kbd> : 버블링 멈춰!
- <kbd>@contextmenu</kbd> : 우클릭 하면 나오는 메뉴
- <kbd>prevent</kbd> : 밑에 함수를 만들지 않아도 e.preventDefault() 적용
- <kbd>once</kbd> : 이벤트를 한 번만 발생시키고 연결 해제
- <kbd>exact</kbd> : 정확하게 일치하는 조합만 실행(복잡한 키보드 조합에서 사용)
