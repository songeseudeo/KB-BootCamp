입력 영역

x :
y :
현재 값 → x: {{ x }}, y: {{ y }}

computed 방식

덧셈 결과 : {{ computedSum }}
computed sum : {{ computedSum }}

- computed는 x, y가 변경될 때만 다시 계산된다.
- 캐시가 있기 때문에 같은 값을 여러 번 사용해도 재계산하지 않는다.

watch 방식

덧셈 결과 : {{ watchSum }}
watch sum : {{ watchSum }}

- watch는 x, y의 변화를 감지해서 직접 sum을 계산한다.
- console.log를 통해 실행 시점을 확인할 수 있다.
