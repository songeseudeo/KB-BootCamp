## 1. 📖 배운 내용
- **주제**: pinia
- **Pinia??**: 여러 컴포넌트 사이에서 공유해야 하는 데이터를 효율적으로 관리하기 위해 사용
- **기본 구성 요소**
  - state: 스토어에서 관리하는 중앙 데이터 (ref, reactive)
  - getters: 계산된 값을 반환 (computed)
  - actions: 상태를 변경하는 로직이나 비동기 작업을 수행하는 함수
- **핵심 함수**
  - defineStore(id, setup): 스토어를 정의하는 함수, 첫 번째 인자로 스토어의 고유 ID를, 두 번째 인자로 스토어의 로직(Setup 함수)을 전달
  - storeToRefs(store): 스토어의 상태(State)를 구조 분해 할당으로 가져올 때, **반응성(Reactivity)**을 유지하기 위해 사용
