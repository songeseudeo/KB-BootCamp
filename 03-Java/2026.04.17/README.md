
# 📅 2026-04-17 학습 기록

## 1. 📖 배운 내용
- **주제**: 객체 정리
- **단어**: 
    - **클래스(Class)**: 객체를 만들기 위한 설계도 (필드, 생성자, 메소드로 구성)
    - **인스턴스(Instance)**: 클래스로부터 생성되어 힙(Heap) 메모리에 배치된 실제 객체
    - **캡슐화(Encapsulation)**: 필드를 `private`으로 보호하고 메소드를 통해 접근하게 하는 것
    - **싱글톤(Singleton)**: 애플리케이션 전체에서 단 하나의 객체만 생성되도록 보장하는 패턴
- **함수**:
    - `Getter / Setter`: private 필드에 안전하게 접근하고 값을 검증하기 위한 메소드
    - `this()`: 생성자 내부에서 자신의 다른 생성자를 호출할 때 사용 (반드시 첫 줄)
    - `getInstance()`: 싱글톤 객체를 외부에서 얻기 위한 정적(static) 메소드
- **중요**: 
    - **static vs Instance**: 정적 멤버는 객체 생성 없이 클래스명으로 접근하며, 모든 객체가 공유함
    - **final 필드**: 초기값이 저장되면 수정할 수 없으며, 상수는 `static final`로 선언함
    - **접근 제한자**: `public` > `protected` > `default` > `private` 순으로 노출 범위를 제어함
