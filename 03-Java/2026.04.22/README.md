# 📅 2026-04-23 학습 기록

## 1. 📖 배운 내용
- **주제**: Java 다형성(Polymorphism) 심화 및 추상 클래스
- **단어**: 
    - **다형성(Polymorphism)**: 사용 방법은 동일하지만 다양한 객체를 이용해 여러 가지 실행 결과가 나오는 성질 [cite: 1076]
    - **자동 타입 변환(Promotion)**: 자식 객체를 부모 타입 변수에 대입할 때 자동으로 일어나는 형변환 [cite: 1078]
    - **추상 클래스(Abstract Class)**: 공통된 특성을 추출해 선언한 클래스로, 실체 클래스의 부모 역할을 수행
    - **추상 메소드(Abstract Method)**: 선언부만 있고 실행 내용(블록)이 없는 메소드
- **함수**:
    - `abstract`: 클래스나 메소드 선언 시 사용하여 추상화를 명시
    - `@Override`: 부모의 추상 메소드를 자식 클래스에서 반드시 구현(재정의)하도록 강제
- **중요**: 
    - **추상 클래스의 목적**: 실체 클래스들의 공통된 필드와 메소드 이름을 통일하여 유지보수 효율 증대
    - **객체 생성 불가**: 추상 클래스는 `new` 연산자를 사용하여 직접 객체를 생성할 수 없음
    - **강제 구현**: 추상 메소드가 포함된 추상 클래스를 상속받으면, 자식 클래스는 반드시 해당 메소드를 재정의해야 함
