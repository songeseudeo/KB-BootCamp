JavaScript 비동기 프로그래밍 실습

1. Promise

        // Promise
        // 비동기 작업의 결과를 나타내는 객체

        // 상태
        // pending   : 작업 진행 중
        // fulfilled : 성공 (resolve)
        // rejected  : 실패 (reject)

        // then()  : 성공 처리
        // catch() : 실패 처리
        // finally(): 항상 실행

        const promise = new Promise((resolve,reject)=>{

        const success = true

        if(success){
        resolve("작업 성공")
        }else{
        reject("작업 실패")
        }

        })

        promise
        .then(result=>console.log(result))
        .catch(error=>console.error(error))
        .finally(()=>console.log("작업 종료"))

실습 : Promise 타이머

3. Fetch API


     Fetch API
     서버에 HTTP 요청을 보내는 JavaScript API
     Promise 기반으로 동작

    fetch(url)
    .then(response=>response.json())
    .then(data=>console.log(data))
    .catch(error=>console.error(error))

     response.json()
     서버에서 받은 JSON 데이터를
     JavaScript 객체로 변환

실습 : 데이터 가져오기

4. Async / Await


    // async / await
    // Promise 코드를 동기 코드처럼 작성하게 해주는 문법

    // async : Promise 반환
    // await : Promise 완료까지 대기

    async function getData(){

        try{

            const response = await fetch(url)

            const data = await response.json()

            console.log(data)

        }catch(error){

            console.error(error)

        }

    }

7. Promise.all


     Promise.all
     여러 비동기 작업을 동시에 실행 (병렬 실행)
     모든 작업이 완료된 후 다음 코드가 실행됨

    Promise.all([
    fetch('/posts/1'),
    fetch('/users/1')
    ])
   
     예
     요청1 시작
     요청2 시작
     ↓
     둘 다 완료
     ↓
     console 실행
